# W-9
Create your own webpage using HTML, CSS and Bootstrap called "index.html". It should have:

A title
Three or more types of headings
Paragraphs
Ordered & unordered lists
Tables
Line breaks & horizontal lines
Comments
Text styles (bold, italic, strike-through, underline)

Additional requirements:

File should be 2–3 screens long
View via browser (localhost or direct open)
Create internal bookmarks (name and href=#)
Split into multiple HTML files and link them
Add external links (e.g., Google)
Move files → links should still work (relative paths)
Add images (inside /images folder)
Use internal CSS (<style> tag)
Then move styles to external CSS file
